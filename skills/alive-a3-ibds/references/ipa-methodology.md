## IPA = Impact Pathway Analysis

IPA คือการวิเคราะห์เส้นทางผลกระทบ (Causal Chain) จาก Input → Impact โดย:
- **Trace** logic chain ของโครงการ
- **Test** plausibility ของแต่ละ link
- **Flag** gaps ที่ต้องแก้
- **Recommend** evidence-based improvements

---

## 6 Modules (Full IPA Analysis)

```
Module A → IDE Classification (who they are)
Module B → IPA Logic Model (what they claim)
Module C → 7 Impact Dimensions (how broad)
Module D → 6 Impact Pathways (how systemic)
Module E → Gap Analysis (what's missing)
Module F → IDEs Funding Readiness (can they get funded)
```

## Module A: IDE Classification

### IDE Type Taxonomy

| Type | Definition | Example |
|------|-----------|---------|
| **Deep Tech** | Novel science/engineering breakthrough | Quantum, gene therapy, fusion |
| **Process Innovation** | New way of doing existing thing | Smart manufacturing, precision ag |
| **Business Model Innovation** | New value/revenue architecture | Platform economy, subscription |
| **Social Innovation** | Addresses social/environmental challenge | Aging care, inclusion, climate |

### IDE Readiness Score (0–10)

| Score | Level | Criteria |
|:-----:|-------|----------|
| 9–10 | Scale-ready | TRL 8+ + product-market fit + revenue |
| 7–8 | Launch-ready | TRL 6–7 + pilot customer + LOI |
| 5–6 | Development | TRL 4–5 + validated concept |
| 3–4 | Early | TRL 2–3 + team formed |
| 0–2 | Pre-IDE | Idea stage |

**Rationale Format:** 2–3 ประโยค อ้างอิงหลักฐานเฉพาะ (TRL, revenue, MOU, etc.)

## Module B: IPA Logic Model

### 6 Elements

| Element | Definition | Example |
|---------|-----------|---------|
| **Inputs** | Resources invested | Budget, staff, equipment |
| **Activities** | What the project does | R&D, pilot, training |
| **Outputs** | Direct products | Publications, patents, prototypes |
| **Short Outcomes** | 0–1 year effects | Adoption, awareness |
| **Long Outcomes** | 2–5 year effects | Behavior change, system change |
| **Impact** | Ultimate benefit | GDP, jobs, environment, well-being |

### Plausibility Scoring (0.0–1.0)

| Score | Star | Criteria |
|:-----:|:----:|----------|
| 0.9–1.0 | ★★★ | Explicit evidence in document |
| 0.7–0.8 | ★★★ | Strongly implied, minor inference |
| 0.5–0.6 | ★★☆ | Plausible but needs evidence |
| 0.3–0.4 | ★☆☆ | Weak — major assumption |
| 0.0–0.2 | ☆☆☆ | Missing or contradicted |

### Plausibility Scoring Method

```
For each element:
  1. Read relevant proposal sections
  2. Identify evidence statements (quote them)
  3. Rate 0.0–1.0 based on directness
  4. Compute star rating
  5. Flag elements < 0.5 as "Critical Gap"
```

### Example Scoring Table

| Element | Evidence Found | Plausibility | Stars |
|---------|----------------|:------------:|:-----:|
| Inputs | "งบ 5M THB + 3 FTE + lab" | 1.0 | ★★★ |
| Activities | "R&D 18 months + pilot" | 0.9 | ★★★ |
| Outputs | "1 patent + 2 papers + prototype" | 0.8 | ★★☆ |
| Short Outcomes | "5 farms adopt" | 0.6 | ★★☆ |
| Long Outcomes | "regional expansion" (implied) | 0.4 | ★☆☆ |
| Impact | "national scale" (no evidence) | 0.2 | ☆☆☆ |

## Module C: 7 Impact Dimensions

ประเมินแต่ละมิติว่า ✅ Present / ⚠️ Partial / ❌ Missing

| # | Dimension | Example Claims |
|:-:|-----------|----------------|
| 1 | **Understanding & Awareness** | ความรู้ใหม่, new knowledge |
| 2 | **Attitudinal** | Behavior change, social norms |
| 3 | **Economic** | GDP, jobs, revenue, cost saving |
| 4 | **Environmental** | CO₂, water, biodiversity |
| 5 | **Health & Well-being** | DALY, mental health, wellness |
| 6 | **Policy (ววน. alignment)** | National KPI, regulation, platform |
| 7 | **Cultural** | Soft power, heritage, identity |

### Scoring Rule
- ✅ **Present (3 pts)**: Explicit + quantified
- ⚠️ **Partial (1 pt)**: Mentioned but not quantified
- ❌ **Missing (0 pts)**: Not addressed

## Module D: 6 Impact Pathways

Rating per pathway: 0–3 ★

| # | Pathway | What to Look For |
|:-:|---------|------------------|
| 1 | **สร้างความรู้คุณภาพสูง** | Publications Q1/Q2, citations, frontier topic |
| 2 | **เสริมทุนมนุษย์** | Training, PhD/MS, skill upgrading |
| 3 | **แพร่กระจายความรู้** | Open access, events, media, policy brief |
| 4 | **ขับเคลื่อนนโยบาย** | Policy input, standard, regulation |
| 5 | **ส่งมอบประโยชน์เชิงพันธกิจ** | University mission, public good |
| 6 | **นำนวัตกรรมเข้าสู่สังคม** | Startup, licensing, commercialization |

### Star Rating
- ★★★ (3): Strong evidence + quantified outcomes
- ★★☆ (2): Mentioned + partial evidence
- ★☆☆ (1): Weak / aspirational
- ☆☆☆ (0): Not addressed

## Module E: Gap Analysis

### Gap Categories

```
Critical Gap  = Plausibility < 0.5  →  MUST FIX
Important Gap = Plausibility 0.5–0.7 →  SHOULD FIX
Minor Gap     = Plausibility 0.7–0.9 →  COULD FIX
```

### Gap Format (per item)
```
Gap: [Element / Dimension]
Current Score: [X/Y] ([Z]%)
Missing Evidence: [specific thing lacking]
Impact on Funding: [High / Medium / Low]
Fix Required: [concrete action]
Timeline: [days/weeks]
Expected Score Gain: +[X] pts
```

### สกสว. Alignment Score (0–10)
Aggregate ของ:
- Platform mapping quality (0–3)
- National KPI linkage (0–3)
- Frontier Tech relevance (0–2)
- Evidence base (0–2)

### Top 3 Recommendations
เรียงตาม **impact × feasibility** — เลือก 3 ข้อที่จะ move the needle มากที่สุด

## Module F: IDEs Funding Readiness

### F1: Sector Eligibility
- Must match 1 of 8 บพข. target sectors (see sector-eligibility.md)
- Alignment level: Strong / Partial / Misaligned

### F2: TRL Gate Check
```
Required: TRL ≥ 4 (baseline)
Required: TRL ≥ 5 for Smart Manufacturing
Required: TRL ≥ 3 for Biotech (special case)
```

### F3: Intermediary Fit
- มี IM ที่เหมาะสม หรือไม่?
- IM track record ใน sector นี้?
- Coaching plan ชัดเจน?

### F4: 5 Innovation Dimensions (ALIVE Part A refs)
Cross-check with Part A scoring:
1. Innovation Capability
2. Business Model
3. Team
4. Market Readiness
5. Financial Sustainability

### F5: IDEs Readiness Score (0–10)
Combined score of F1–F4 with weights:
- F1 Sector: 20%
- F2 TRL: 20%
- F3 IM Fit: 20%
- F4 5 Dimensions: 40%

**Decision Rules:**
- Score 8–10: Apply now
- Score 6–7: Apply after 30-day prep
- Score 4–5: Apply after 60-day IM coaching
- Score < 4: Not ready — fundamental gaps

## Full Report Structure (8–12 pages)

```
Page 1:    Executive Summary + Score Dashboard
Page 2:    IDE Classification (Module A)
Page 3–4:  IPA Logic Model + Plausibility (Module B)
Page 5:    7 Impact Dimensions (Module C)
Page 6:    6 Impact Pathways (Module D)
Page 7–8:  Gap Analysis + Recommendations (Module E)
Page 9:    Funding Readiness (Module F)
Page 10:   SROI Forecast Summary (link to A4)
Page 11–12: Appendix — Literature Evidence (APA)
```

## Quality Rules (IBDS Deliverable)

1. **Evidence-backed:** ทุก claim ต้องมี citation หรือ document reference
2. **Quantified:** ใส่ตัวเลขทุกที่ที่ทำได้
3. **Visual:** มี chart/table อย่างน้อย 5 จุด
4. **Actionable:** Recommendations ต้อง concrete + time-bound
5. **Bilingual:** Thai headings + English technical terms (default)
