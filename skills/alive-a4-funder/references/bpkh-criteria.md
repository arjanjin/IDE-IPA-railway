## Funding Decision Matrix (Combined 4-Part + SROI)

| Condition | Decision | Budget Allocation |
|-----------|----------|-------------------|
| A≥24, B≥18, C≥22, D≥22, SROI≥2.5x | **STRONGLY FUND** | 100% ของที่ขอ |
| A≥16, B≥13, C≥17, D≥17, SROI≥1.5x | **FUND** | 80–100% |
| A≥9, B≥8, C≥12, D≥12, SROI≥1.0x | **CONDITIONAL FUND** | 50–80% + KPI gate |
| Any threshold failed OR SROI < 1.0x | **DO NOT FUND** | 0% |

**Minimum Threshold (ALL must pass):**
```
A ≥ 9  AND  B ≥ 8  AND  C ≥ 12  AND  D ≥ 12  AND  Pessimistic SROI ≥ 1.0x
```

## Special Cases & Conditional Paths

### Case 1: High SROI + Low IDE
```
Condition: D ≥ 22, A < 16
Decision:  FUND + IDE Development Support
Support:   Mentoring, Training, Partnership facilitation
Timeline:  6-month IDE uplift plan
```

### Case 2: High IDE + Low SROI
```
Condition: A ≥ 24, D < 12
Decision:  CONDITIONAL — Fix SROI model within 60 days
Required:  Revised Theory of Change + stronger proxies
Gate:      Re-score D ≥ 17 before first disbursement
```

### Case 3: Research-only (Low IDE+SROI)
```
Condition: B ≥ 18, A < 16, D < 17
Decision:  Research funding only (not IDE program)
Redirect:  Forward to TSRI/วช. research-only track
```

### Case 4: Pathway Weak (Low C)
```
Condition: D ≥ 17 but C < 12
Decision:  CONDITIONAL — Fix Theory of Change first
Reason:    SROI ที่ดีต้องมี Pathway Logic รองรับ
Gate:      Re-score C ≥ 17 before fund commitment
```

### Case 5: ววน. Misalignment
```
Condition: B10 (ววน. Alignment) = 0
Decision:  CONDITIONAL — Fix Platform alignment first
Required:  Map to ≥1 Platform + ≥1 National KPI
Gate:      B10 ≥ 1 before funding
```

## Approval Workflow (บพข.)

```
Step 1: Submit Proposal + A1 Scorecard
Step 2: A3 Deep Analysis (external reviewer)
Step 3: A4 SROI Report + Policy Brief
Step 4: Internal Committee Review (บพข.)
Step 5: Funding Decision
Step 6: Contract + Disbursement (tranche-based)
Step 7: Mid-term Review (A2 IM report)
Step 8: Final Evaluation (Evaluative SROI)
```

## Red Flags (Automatic Do-Not-Fund)

🚩 ตรวจเจอ red flag ต่อไปนี้ → **DO NOT FUND**

1. **SROI < 1.0x** ใน Pessimistic scenario
2. **Any Part threshold failed** (A<9, B<8, C<12, D<12)
3. **ววน. Platform misalignment** (B10=0 และไม่ยอมปรับ)
4. **No industry partner** สำหรับโครงการ Commercialization
5. **Conflict of interest** ไม่ declared
6. **Duplicate funding** (ขอทุนซ้ำจาก PMU อื่น)
7. **PI track record ต่ำ** (< 3 ปี หรือ no publication)
8. **Ethical issue** (ไม่ผ่าน IRB/IACUC เมื่อจำเป็น)
